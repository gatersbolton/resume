2025 年 07 月 - 至今

研究生课题组科研成果 Web 应用

集成校内地质力学课题组的多项科研 Python 算法为三层 Web 应用，前端 Vue3 + Element Plus，后 端 WebFlux 网关 与 MyBatis 用户模块，FastAPI 微服务 承载计算与可视化，支持演示数据、一键计算、进度查询和结果下载。

  • 负责整体架构与落地：抽象计算/进度/结果/下载/可视化接口，网关转发长耗时任务并设计异步协议； 前端四步工作流、进度轮询组件与“预览/最终”双模渲染，显著提升交互体验。
  • 提供演示数据与一键计算，以便快速演示科研算法的业务价值；
  • 设计下载与可视化接口，确保科研结果便于分享与复现。

2024 年 03 月 - 2024 年 09 月

Spring Boot + Redis 仿大众点评

参考大众点评的业务流程，用 Spring Boot 落地店铺、点评、签到、优惠券等模块， 用 Redis 处理登录态、热点店铺缓存、地理位置搜索与秒杀队列， 搭配 分布式锁 确保库存一致性，消息异步用 Kafka 解耦。

  • 登录鉴权与七天签到打卡，利用 Redis Bitmap 与 HyperLogLog，吸顶榜单支持高并发计数；
  • 店铺缓存穿透/雪崩/击穿防护：空值缓存、逻辑过期 + 异步重建、热点 Key 互斥锁；
  • 秒杀优惠券：分布式锁 + 队列异步下单，保序消费校验库存，并记录下单链路日志；
  • JVM 与线程池参数按压测调优，结合监控定位 GC、慢查询和缓存命中率异常。

  • 封装统一大模型调用 SDK，支持接口查询、参数示例生成与错误原因归纳，服务内部研发同学。
